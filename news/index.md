# Changelog

## AccumulatR 0.1.0

- First public release.
- Added tools for defining evidence-accumulation models with
  accumulators, pools, response rules, mixture components, and chained
  onsets.
- Added simulation helpers for generating behavioral data from model
  specifications.
- Added likelihood tools for evaluating observed response-time data
  under candidate parameter values.
- Added pkgdown articles and reference documentation for the main
  modeling workflow.
